# Equipped Website

Marketing site for [Equipped](https://tryequipped.com) — IT handled for growing companies.

## Stack

Static HTML/CSS/JS. No build step required.

## Deploy

Push to GitHub and connect to Cloudflare Pages, GitHub Pages, or Vercel. Point `tryequipped.com` DNS to the host.

## Structure

```
index.html          # Landing page
llms.txt            # LLM-readable company summary
logos/normalized/   # Client logos (PNG, normalized)
```
